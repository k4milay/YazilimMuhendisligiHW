# 📊 Git Mini Dashboard

Bu proje, Git ve GitHub üzerinde takım çalışması (collaboration) adımlarını uygulamalı olarak öğrenmek ve **Merge Conflict (Çakışma)** krizlerini yönetebilmek amacıyla geliştirilmiştir.

## 🚀 Proje İçeriği

* **`main.py`:** Sistemin ana çalıştırıcı dosyasıdır. Diğer modülleri birleştirir.
* **`weather.py`:** Bağımsız olarak geliştirilen hava durumu modülü.
* **`news.py`:** Bağımsız olarak geliştirilen haber modülü.

## 👨‍💻 Geliştiriciler

* **Kamil:** Proje deposunu kuran, `weather.py` modülünü geliştiren ve sistemin final dokunuşlarını (çıktılarını) ayarlayan geliştirici.
* **Ersin:** `news.py` modülünü geliştiren ve projede bilerek tetiklenen "Merge Conflict" (çakışma) krizini başarıyla çözerek kodları birleştiren geliştirici.

## 📌 Bu Ödevde Neler Öğrendik?

1.  GitHub üzerinde ortak repository kullanımı ve `git clone` işlemi.
2.  Farklı dosyalarda paralel geliştirme yapma (`git add`, `git commit`, `git push`).
3.  Aynı dosya üzerinde çalışırken ortaya çıkan **Merge Conflict** durumunu anlama.
4.  Çakışmaları manuel olarak kod editöründe çözme ve projeyi senkronize etme (`git pull`).
